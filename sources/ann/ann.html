Artificial Neural Networks (ANN)¶

This sub-module contains neural networks and all related stuff. So far it is implemented only MLP (Multi-layer Perceptron).

# Examples

## MLP - Multi-layer Perceptron

Following code will create MLP network for five inputs with two hidden layers (three nodes in each hidden layer).

>>> import padasip as pa
>>> pa.ann.NetworkMLP([3,3], 5)
<padasip.ann.mlp.NetworkMLP instance at 0xb7200bac>

Multi-layer Perceptron (MLP)¶

In this module is stored everything related to Multi-layer perceptron (MLP). This neural network can be used for classification and regression.
