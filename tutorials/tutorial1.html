PADASIP Module: Noise Cancelation, Identification and Prediction with Adaptive Filters¶

This tutorial explains how to perform various tasks with adaptive filters through several examples.

Lets start with importing padasip. In the following examples we will also use numpy and matplotlib.

Let us consider a time series $\textbf{u}$

If the series is only an input of the adaptive filter, the input matrix can be a so called Delay Embedded Matrix (DEM). The construction of the input matrix can be simply created as

x = pa.input_from_history(u, 4)

Noise Cancelation (Example with RLS Filter)¶

Consider a sin like natured signal $\textbf{u}$ contaminated with noise $\textbf{v}$. This results in a measured signal $\textbf{d}$. The goal is prediction/reconstruction of signal $\textbf{u}$ only from signal $\textbf{d}$. For the evaluation of noise cancelation performance we will use the MSE (mean-squared-error).

In [7]:
# signals creation: u, v, d
N = 1000
n = 15
u = np.sin(np.arange(0, N/10., N/10000.))
v = np.random.random(N) - 0.5
d = u + v

# filtering
x = pa.input_from_history(d, n)[:-1]
d = d[n:]
u = u[n:]
y, e, w = pa.rls_filter(d, x, mu=0.95)

# error estimation
MSE_d = np.dot(u-d, u-d) / float(len(u))
MSE_y = np.dot(u-y, u-y) / float(len(u))

# results
plt.figure(figsize=(15,6))
plt.plot(u, "r:", linewidth=4, label="original")
plt.plot(d, "b", label="noisy, MSE: {}".format(MSE_d))
plt.plot(y, "g", label="filtered, MSE: {}".format(MSE_y))
plt.xlim(800,900)
plt.legend()
plt.show()

According to this result, the adaptive filter can decrease the MSE caused by noise approximately by half.

System Identification (Example with NLMS Filter)¶

Consider a system output contaminated with noise $\textbf{v}$

$d(k) = 2 x_1(k) + 0.1 x_2(k) - 4 x_3(k) + 0.5 x_4(k) + v(k)$

where $x_i(k)$ is $i$-th channel (column) in $k$ sample (row) of the input matrix $\textbf{X}$ (random numbers in this example) - in this case, the input of the system is the same as the input of the filter. The noise $\textbf{v}$ is random numbers in range from $-0.5$ to $+0.5$. Because the scale of the data is small, we can use a high learning rate $\mu=0.5$

In [8]:
# creation of x and d
N = 500
x = np.random.random((N, 4))
v = np.random.random(N) - 0.5
d = 2*x[:,0] + 0.1*x[:,1] - 4*x[:,2] + 0.5*x[:,3] + v

# identification
y, e, w = pa.nlms_filter(d, x, mu=0.5)

# show results
plt.figure(figsize=(15,9))
plt.subplot(211);plt.title("Adaptation");plt.xlabel("samples - k")
plt.plot(d,"b", label="d - target")
plt.plot(y,"g", label="y - output");plt.legend()
plt.subplot(212);plt.title("Filter error");plt.xlabel("samples - k")
plt.plot(abs(e),"r", label="abs(e) - error");plt.legend()
plt.tight_layout()
plt.show()
print("And the resulting coefficients are: {}".format(w))

And the resulting coefficients are: [ 2.09026265  0.17186359 -3.99786945  0.6219337 ]

Prediction (Example with RLS Filter)¶

Consider measured data produced by the following system

$d(k) = a_1 u(k) + a_2 u(k-1) + a_3 u(k-2) + a_4 u(k-3)$

For $d(0)$ we need initial conditions $[u(-3), u(-2), u(-1), u(0)]$. In other words, for $u$ of size $100$ the maximal size of $d$ is $97$.

In [9]:
# creation of u, x and d
N = 100
u = np.random.random(N)
d = np.zeros(N)
for k in range(3, N):
    d[k] = 2*u[k] + 0.1*u[k-1] - 4*u[k-2] + 0.5*u[k-3]
d = d[3:]

# identification
x = pa.input_from_history(u, 4)
y, e, w = pa.rls_filter(d, x, mu=0.1)

# show results
plt.figure(figsize=(15,9))
plt.subplot(211);plt.title("Adaptation");plt.xlabel("samples - k")
plt.plot(d,"b", label="d - target")
plt.plot(y,"g", label="y - output");plt.legend()
plt.subplot(212);plt.title("Filter error");plt.xlabel("samples - k")
plt.plot(abs(e),"r", label="abs(e) - prediction error");plt.legend()
plt.tight_layout()
plt.show()

From the above figure it is possible to see, that the prediction error decreases with new samples. At the end, the error is really low.
